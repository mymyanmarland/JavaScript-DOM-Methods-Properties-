Methods & Properties အပြည့်အစုံ

လေ့လာရန် အဆင်သင့်ဖြစ်ပြီလား?

DOM ဆိုတာ ဘာလဲ?

DOM (Document Object Model) ဆိုတာ HTML document ကြီးတစ်ခုလုံးကို JavaScript ကနေ လှမ်းပြီး ထိန်းချုပ်ပြင်ဆင်လို့ရအောင် Browser က တည်ဆောက်ပေးထားတဲ့ Object Tree ဖြစ်ပါတယ်။

DOM ကိုအသုံးပြုပြီး HTML elements တွေကို အသစ်ထည့်တာ၊ ဖျက်တာ၊ ပြင်တာ၊ CSS style တွေပြောင်းတာ အကုန်လုပ်လို့ရပါတယ်။

DOM Tree Structure

HTML document တစ်ခုလုံးကို သစ်ပင် (Tree) ပုံစံ ဖွဲ့စည်းထားပါတယ်။

  • Document - အမြစ် (Root)
  • Elements - အကိုင်းအခက်များ (html, head, body, div, p)
  • Text - အရွက်များ (စာသားများ)

Window vs Document

Window Object

Browser ရဲ့ ပြတင်းပေါက် (Tab) ကို ကိုယ်စားပြုပါတယ်။ Global object ဖြစ်ပါတယ်။

Document Object

Window ထဲမှာပြနေတဲ့ HTML စာမျက်နှာကို ကိုယ်စားပြုပါတယ်။ DOM ရဲ့ အစမှတ် (Entry point) ဖြစ်ပါတယ်။

အပိုင်း (၁): Elements များကို ရွေးချယ်ခြင်း

HTML tag တွေကို JavaScript ထဲရောက်အောင် လှမ်းဖမ်းတဲ့ Methods များ

1. getElementById()

HTML ရဲ့ id attribute ကို အသုံးပြုပြီး Element ကို လှမ်းခေါ်တဲ့ Method ပါ။ တစ်ခုတည်းသော element ကိုပဲ return ပြန်ပေးပါတယ်။


// id="title" ရှိသော element ကို ရယူခြင်း
const myTitle = document.getElementById('title');
console.log(myTitle);

2. getElementsByClassName()

class နာမည်တူတဲ့ Elements တွေ အကုန်လုံးကို စုပြီး HTMLCollection (Array-like object) အနေနဲ့ ပြန်ပေးပါတယ်။


// class="items" ရှိသော element အားလုံးကို ရယူခြင်း
const items = document.getElementsByClassName('items');
// ပထမဆုံး item ကို ယူချင်လျှင်
console.log(items[0]);

3. getElementsByTagName()

HTML tag နာမည် (ဥပမာ - p, div, li) တွေကို အသုံးပြုပြီး elements တွေကို ရှာဖွေပါတယ်။


// <p> tag အားလုံးကို ရယူခြင်း
const paragraphs = document.getElementsByTagName('p');
console.log(paragraphs.length); // p ဘယ်နှခုရှိလဲ ကြည့်ရန်

4. querySelector()

CSS Selector (ဥပမာ - .class, #id, tag) တွေကို အသုံးပြုပြီး ရှာဖွေပါတယ်။ တွေ့တဲ့အထဲက ပထမဆုံး တစ်ခုကိုသာ ပြန်ပေးပါတယ်။ အသုံးအများဆုံးပါ။


const firstItem = document.querySelector('.item'); // class
const title = document.querySelector('#title'); // id
const firstBtn = document.querySelector('button'); // tag

5. querySelectorAll()

CSS Selector နဲ့ ကိုက်ညီတဲ့ Elements အားလုံးကို NodeList အနေနဲ့ ပြန်ပေးပါတယ်။ forEach နဲ့ ပတ်လို့ရပါတယ်။


const allItems = document.querySelectorAll('.list-item');
allItems.forEach(item => {
    item.style.color = 'red';
});

HTMLCollection vs NodeList

  • HTMLCollection: getElementsBy... တွေက ပြန်ပေးတယ်။ Array method တွေ (forEach) သုံးလို့မရဘူး။ Live ဖြစ်တယ်။
  • NodeList: querySelectorAll က ပြန်ပေးတယ်။ forEach သုံးလို့ရတယ်။ Static ဖြစ်တယ်။

အပိုင်း (၂): DOM Traversing

Element တစ်ခုကနေ အထက်၊ အောက်၊ ဘေးဘယ်ညာ ကို သွားလာခြင်း Property များ

6. parentNode

လက်ရှိ Element ရဲ့ အထက်က မိဘ (Parent) Node ကို ပြန်ပေးပါတယ်။


const child = document.querySelector('.child');
console.log(child.parentNode); 
// ၎င်း၏ အထက် div ကို ပြပေးမည်

7. parentElement

parentNode နဲ့ အတူတူပါပဲ၊ ဒါပေမယ့် သူက HTML Element အစစ်ဖြစ်မှသာ ပြန်ပေးပါတယ်။ (များသောအားဖြင့် parentElement ကိုပဲ သုံးကြပါတယ်)


const listItem = document.querySelector('li');
const ulElement = listItem.parentElement;

8. children

လက်ရှိ Element ရဲ့ အောက်မှာရှိတဲ့ သားသမီး (Child) HTML elements အားလုံးကို HTMLCollection အနေနဲ့ ပြန်ပေးပါတယ်။ Text node တွေ မပါဝင်ပါ။


const list = document.querySelector('ul');
console.log(list.children); // 
  •  အားလုံးကို ပြမည်

    • 9. childNodes

    Element အောက်မှာရှိတဲ့ Node တွေအားလုံးကို ပြပေးပါတယ်။ (Space တွေ၊ Enter ခေါက်ထားတဲ့ Text Node အလွတ်တွေပါ အကုန်ပါဝင်ပါတယ်)

    
const list = document.querySelector('ul');
console.log(list.childNodes); // NodeList ကို ပြမည်

    10. firstElementChild

    မိဘ Element တစ်ခုရဲ့ ပထမဆုံးသော Child HTML Element ကိုသာ ရယူပေးပါတယ်။

    
const list = document.querySelector('ul');
const firstLi = list.firstElementChild;
firstLi.style.fontWeight = 'bold';

    11. lastElementChild

    မိဘ Element ရဲ့ နောက်ဆုံး Child HTML Element ကို ရယူပေးပါတယ်။

    
const list = document.querySelector('ul');
const lastLi = list.lastElementChild;
lastLi.style.color = 'blue';

    12. nextElementSibling

    လက်ရှိ Element ရဲ့ အဆင့်တူ (Sibling) ဖြစ်တဲ့ နောက်တစ်ခုကို ဆက်ယူပေးပါတယ်။

    
const item2 = document.querySelector('.item-2');
const item3 = item2.nextElementSibling;

    13. previousElementSibling

    လက်ရှိ Element ရဲ့ အဆင့်တူဖြစ်တဲ့ အရှေ့ကတစ်ခုကို ရယူပေးပါတယ်။

    
const item2 = document.querySelector('.item-2');
const item1 = item2.previousElementSibling;

    အပိုင်း (၃): Elements များကို ပြင်ဆင်ခြင်း

    စာသားတွေပြောင်းမယ်၊ HTML တွေ အစားထိုးမယ်

    14. innerHTML

    Element အတွင်းက HTML code တွေကို ရယူဖို့ (သို့) အသစ်ထည့်သွင်းဖို့ သုံးပါတယ်။ HTML tag တွေကို သိပါတယ်။

    
const box = document.querySelector('.box');
box.innerHTML = '
    Hello World
    '; // H2 tag အလုပ်လုပ်မည်

    15. textContent

    Element အတွင်းက စာသားသက်သက်ကိုသာ ပြင်ဆင်/ရယူ ပါတယ်။ HTML tag တွေ ထည့်ရေးရင် စာသားအနေနဲ့ပဲ ပေါ်ပါမယ်။ (Security အတွက် ပိုကောင်းပါတယ်)

    
const box = document.querySelector('.box');
box.textContent = 'Hello World';

    16. innerText

    textContent နဲ့ ဆင်တူပါတယ်။ သို့သော် သူက CSS နဲ့ ဖျောက်ထားတဲ့ (display: none) စာသားတွေကို မဖတ်ပါဘူး။ Browser မှာ မြင်ရတဲ့အတိုင်းပဲ ယူပါတယ်။

    
const para = document.querySelector('p');
console.log(para.innerText);

    အပိုင်း (၄): Elements အသစ်ဖန်တီးခြင်း

    JavaScript ဖြင့် HTML tag များကို အသစ်ဖန်တီးခြင်း

    17. createElement()

    HTML element (tag) အသစ်တစ်ခုကို ဖန်တီးပေးပါတယ်။ Document ထဲကိုတော့ မရောက်သေးပါဘူး၊ Memory ထဲမှာပဲ ရှိပါသေးတယ်။

    
// 
     အသစ်ဖန်တီးခြင်း
const newDiv = document.createElement('div');
newDiv.className = 'my-new-div';

    18. createTextNode()

    HTML tag မပါတဲ့ စာသားသက်သက် (Text Node) အသစ်တစ်ခု ဖန်တီးပေးပါတယ်။

    
const myText = document.createTextNode('This is new text');
// newDiv.appendChild(myText);

    အပိုင်း (၅): Elements များကို ထည့်သွင်းခြင်း

    ဖန်တီးထားသော Element များကို နေရာချခြင်း

    19. appendChild()

    Element တစ်ခုရဲ့ နောက်ဆုံးအဆုံး (Last child) အဖြစ် သွားရောက် ထည့်သွင်းပေးပါတယ်။ Node တစ်ခုတည်းသာ ထည့်လို့ရပါတယ်။

    
const list = document.querySelector('ul');
const newLi = document.createElement('li');
newLi.textContent = 'New Item';

list.appendChild(newLi); // အောက်ဆုံးမှာ သွားပေါ်မည်

    20. append()

    appendChild နဲ့ တူပေမယ့် သူက Node တွေရော၊ Text String တွေပါ တစ်ပြိုင်နက်တည်း အများကြီး ထည့်လို့ရပါတယ်။ (ခေတ်သစ် Method ပါ)

    
const box = document.querySelector('.box');
box.append('Some text', document.createElement('p'));

    21. prepend()

    Element ရဲ့ ပထမဆုံးနေရာ (First child) အဖြစ် ထိပ်ဆုံးကနေ သွားရောက် ထည့်သွင်းပေးပါတယ်။

    
const list = document.querySelector('ul');
const newLi = document.createElement('li');
newLi.textContent = 'First Item';

list.prepend(newLi); // ထိပ်ဆုံးမှာ သွားပေါ်မည်

    22. insertBefore()

    Element တစ်ခုရဲ့ အရှေ့တိတိမှာ နေရာချချင်တဲ့အခါ သုံးပါတယ်။ (ထည့်ချင်သောအရာ, ဘယ်အရာရှေ့မှာလဲ)

    
const parent = document.getElementById('parent');
const newNode = document.createElement('div');
const referenceNode = document.getElementById('child2');

parent.insertBefore(newNode, referenceNode);

    23. insertAdjacentHTML()

    HTML string တွေကို သတ်မှတ်ထားတဲ့ နေရာ ၄ နေရာထဲက တစ်နေရာမှာ အတိအကျ ထည့်ပေးပါတယ်။ အရမ်းမြန်ပါတယ်။

    
const box = document.querySelector('.box');
// 'beforebegin', 'afterbegin', 'beforeend', 'afterend'
box.insertAdjacentHTML('beforeend', '
    New Para
    ');

    အပိုင်း (၆): Elements များကို ဖယ်ရှားခြင်း

    မလိုအပ်တော့တဲ့ HTML တွေကို ဖြုတ်ချမယ်

    24. removeChild()

    မိဘ Element ကနေတစ်ဆင့် သားသမီး Element ကို ရွေးချယ်ပြီး ဖယ်ရှားပါတယ်။

    
const list = document.querySelector('ul');
const itemToRemove = list.firstElementChild;

list.removeChild(itemToRemove);

    25. remove()

    လက်ရှိ Element ကိုယ်တိုင်ကို တိုက်ရိုက် ဖျက်ပစ်လိုက်ပါတယ်။ (ES6 Method ပါ၊ သုံးရတာ ပိုလွယ်ကူပါတယ်)

    
const alertBox = document.querySelector('.alert');
alertBox.remove(); // ပျောက်သွားမည်

    26. replaceChild()

    ရှိပြီးသား Element အဟောင်းကို အသစ်တစ်ခုနဲ့ အစားထိုးပါတယ်။ (အသစ်, အဟောင်း)

    
const parent = document.getElementById('parent');
const newChild = document.createElement('span');
const oldChild = document.getElementById('old');

parent.replaceChild(newChild, oldChild);

    အပိုင်း (၇): Attributes များကို အသုံးပြုခြင်း

    id, src, href စသည်တို့ကို ပြင်ဆင်ခြင်း

    27. getAttribute()

    Element ရဲ့ Attribute တန်ဖိုး (Value) ကို လှမ်းယူပါတယ်။

    
const img = document.querySelector('img');
const imgSrc = img.getAttribute('src');
console.log(imgSrc);

    28. setAttribute()

    Attribute အသစ်တစ်ခု ထည့်ပေးတာ သို့မဟုတ် ရှိပြီးသားကို တန်ဖိုးအသစ် ချိန်းပေးတာ လုပ်ပါတယ်။

    
const link = document.querySelector('a');
link.setAttribute('href', 'https://google.com');
link.setAttribute('target', '_blank');

    29. removeAttribute()

    မလိုအပ်တော့တဲ့ Attribute ကို ဖျက်ပစ်ပါတယ်။

    
const input = document.querySelector('input');
input.removeAttribute('disabled'); // စာပြန်ရိုက်လို့ရသွားမည်

    30. hasAttribute()

    ဒီ Element မှာ သတ်မှတ်ထားတဲ့ Attribute ရှိ/မရှိ (true / false) ပြန်စစ်ပေးပါတယ်။

    
const btn = document.querySelector('button');
if(btn.hasAttribute('disabled')) {
    console.log('Button is locked!');
}

    အပိုင်း (၈): CSS နှင့် Class များ ပြင်ဆင်ခြင်း

    DOM ကိုအသုံးပြုပြီး ဒီဇိုင်းများ ချက်ချင်းပြောင်းလဲခြင်း

    31. element.style (Inline Styles)

    CSS properties တွေကို JavaScript ကနေ တိုက်ရိုက်လှမ်းပြင်တာပါ။ background-color ကို backgroundColor လို့ CamelCase ရေးရပါတယ်။

    
const box = document.querySelector('.box');
box.style.backgroundColor = 'blue';
box.style.fontSize = '20px';
box.style.display = 'none'; // ဖျောက်ထားမည်

    32. className Property

    Element ရဲ့ Class တွေ အားလုံးကို String တစ်ခုအနေနဲ့ ရယူတာ၊ ပြန်သတ်မှတ်တာ လုပ်လို့ရပါတယ်။

    
const div = document.querySelector('div');
div.className = 'box active highlight'; // အဟောင်းတွေ ပျက်သွားနိုင်သည်

    33. classList.add()

    Class အသစ်တစ်ခု (သို့) အများကြီးကို Element ထဲ ထပ်ထည့်ပေးပါတယ်။ (ရှိပြီးသား Class တွေ မပျက်ပါ)

    
const modal = document.querySelector('.modal');
modal.classList.add('show', 'animate');

    34. classList.remove()

    သတ်မှတ်ထားတဲ့ Class ကို ဖယ်ရှားပေးပါတယ်။

    
const modal = document.querySelector('.modal');
modal.classList.remove('show'); // ဖျောက်လိုက်မည်

    35. classList.toggle()

    အရမ်းအသုံးဝင်ပါတယ်။ Class မရှိရင် ထည့်ပေးမယ်၊ ရှိနေရင် ဖြုတ်ပေးပါမယ်။ (On/Off ခလုတ်တွေမှာ သုံးပါတယ်)

    
const btn = document.querySelector('.theme-btn');
btn.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
});

    36. classList.contains()

    ဒီ Class ပါဝင်နေသလား ဆိုတာကို true / false ပြန်ပေးပါတယ်။ စစ်ဆေးချင်တဲ့အခါ သုံးပါတယ်။

    
const menu = document.querySelector('.menu');
if(menu.classList.contains('active')) {
    console.log('Menu ဖွင့်ထားတယ်');
}
